# Personal Portfolio Website

This project is my personal portfolio website, developed as part of my bachelor's program at Epicode University. The website is currently live and accessible at [sebastiantanase.it](https://sebastiantanase.it). It showcases my skills, projects, and professional journey, reflecting my expertise in web development.

---

## 🚀 Features

- **Responsive Design**: Built using Bootstrap for a mobile-first, fully responsive layout.
- **Project Showcases**: Detailed descriptions and visuals of my past and current projects.
- **About Me Section**: Information about my background, education, and aspirations.
- **Contact Form**: Functional form for inquiries or collaboration opportunities.

---

## 🛠️ Technologies Used

- **HTML5**: Structure of the website.
- **CSS3**: Styling and layout.
- **JavaScript**: Interactivity and dynamic content.
- **Bootstrap**: Framework for responsive design.
- **Git**: Version control for project management.

---

## 📝 Installation and Setup

1. Clone the repository to your local machine:
   ```bash
   git clone https://github.com/TanaseSebastian/projectEpicode.git
   ```
2. Navigate to the project directory:
   ```bash
   cd projectEpicode
   ```
3. Open `index.html` in your browser to view the website.
